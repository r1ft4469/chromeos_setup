Chrome OS Setup Commands

====

1. Get Developer Mode
```
ESC+F3+Power
At Recovery Screen
CTRL+D
```
2. Run Inital Setup
3. Set Password
```
chromeos-setdevpassword
```
4. Setup
```
wget -q -O - https://raw.githubusercontent.com/r1ft4469/chromeos_setup/master/env_setup.sh | bash
```
5. Restart
```
chrome://inducebrowsercrashforrealz
```
